ENTREGABLE 2 - Simulador de Turnos para Estudio de Tatuajes
Alumno: FACUNDO FARACI
COMISION: 80805
Curso: JavaScript

Descripción:
Este proyecto simula un generador de turnos para un estudio de tatuajes. El usuario puede cargar su nombre, seleccionar la zona a tatuar, elegir una fecha y hora, y dejar una breve descripción del diseño que quiere.

Tecnologías usadas:
- HTML5
- CSS3 + Bootstrap 5
- JavaScript (manejo de DOM, eventos, localStorage)

Funcionalidades principales:
- Formulario para ingresar turnos personalizados.
- Los turnos se guardan en un array de objetos y se visualizan en el DOM.
- Se pueden eliminar turnos individualmente.
- Toda la información se guarda automáticamente en localStorage para mantener los datos aunque se recargue la página.
- Estilos propios combinados con Bootstrap para que la app sea responsive y visualmente clara.
